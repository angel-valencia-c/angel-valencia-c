I'm passionate about building practical, efficient software solutions — from backend APIs to interactive web interfaces. I enjoy tackling real-world problems through code and I'm constantly expanding my skill set across languages, frameworks, and infrastructure tools.

- <div>

    ... <strong>Proyecto Jomicel</strong> — a multiplatform software solution designed to streamline internal operations for a local business. The project involves building a relational database from scratch, developing a PHP-based REST backend, and crafting a React frontend with a responsive UI. I'm handling the full stack: from data modeling and query optimization in MySQL, to component architecture in React and containerization with Docker to ensure consistent environments across development and production.
